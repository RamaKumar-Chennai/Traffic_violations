# Traffic Violations Insight System 🚦

## 📌 Overview
A data analytics project that transforms ~1M raw traffic violation records into actionable insights using **Python, SQL, EDA, and Streamlit**.  
Focus: **Data cleaning, preprocessing, visualization, and interactive dashboards** for urban safety analysis.

## 🎯 Objectives
- Clean & preprocess noisy, inconsistent records  
- Perform structured EDA (common violations, hotspots, demographics, time trends, vehicle types)  
- Build an interactive Streamlit dashboard with filters, charts, and heatmaps  

## 💡 Business Use Cases
- Improve road safety  
- Identify high‑risk zones  
- Detect repeat offenders  
- Support patrol allocation & policy decisions  

## 📊 Skills Gained
- Python Programming  
- SQL  
- Data Cleaning & Pre‑processing  
- Streamlit  
- Visualization (Matplotlib / Seaborn / Plotly)
